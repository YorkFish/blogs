    4. Export¶

    方法¶

    • 第一次导出，使用命令：mkdocs build
    • 第一次以后，使用命令：mkdocs build --clean

    • 若操作无误，命令行会显示类似如下信息

      1
2
      > INFO    -  Cleaning site directory
> INFO    -  Building documentation to directory: D:\my-project\site

    • 然后，项目文件夹下会新增一个名为 site 的文件夹，里面是生成的静态网页
