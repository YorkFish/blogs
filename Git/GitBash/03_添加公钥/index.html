    在 GitHub 上添加公钥

    1. 新建 SSH Key

    1. 点击右上角的头像

    2. 点击 Settings

    3. 选择 SSH and GPG keys

    4. 点击右上角的 New SSH Key

    2. 填入信息

    1. 在 Title 处输入自己想取的名字，如 win10、Xubuntu16 等
    2. 在 Key 处输入之前复制的**公钥**
      • 如果第一次不成功，可以再试一次
      • 有时不成功可能是末尾多了个回车的缘故
    3. 点击 Add SSH Key
