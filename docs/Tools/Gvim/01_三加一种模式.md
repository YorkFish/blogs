## 4. “可视” 模式

- 这个模式是基于“编辑模式”的
- 按 <kbd>v</kbd> 进入 **“可视”** 模式——“仿鼠标选择”
- 按 <kbd>Ctrl</kbd> + <kbd>v</kbd> 进入 **“可视-块”** 模式——“列选择”
- 按 <kbd>Shift</kbd> + <kbd>v</kbd> 进入 **“可视-块”** 模式——“行选择”
