# 转义字符

## 1. 用法

- 与许多语言一样，*Markdown* 中的转义字符也是 `\`
- 用的时候，在需要转义的字符前加上反斜杠即可

## 2. 说明

- 下表中的符号使用时需要转义

    | 序号 | 符号 | 名称 |
    | :---: | :---: | :---: |
    | 01 | \# | 井号 |
    | 02 | \* | 星号 |
    | 03 | \- | 减号 |
    | 04 | \+ | 加号 |
    | 05 | \` | 反引号 |
    | 06 | \\ | 反斜杠 |
    | 07 | \_ | 下划线 |
    | 08 | \{\} | 大括号 |
    | 09 | \[\] | 中括号 |
    | 10 | \(\) | 小括号 |
    | 11 | \! | 英文叹号 |
    | 12 | \. | 英文句号 |
