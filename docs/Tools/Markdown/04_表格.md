# 表格

## 1. 用法

- 标题栏 + 停靠设置 + 具体内容
- 形如

        |  |  |  |				——标题栏
        | :--- | :---: | ---: |	——停靠设置
        |  |  |  |				—— 具体内容
        ...

## 2. 说明

- 停靠效果有三种
    - `:---` ——左对齐
    - `:---:` ——居中
    - `---:` —— 右对齐
- `-` 必须大于等于 *3* 个
- 只用 `-` 也行，但需要计算个数，默认为左对齐
- 有的软件，表格上下各空一行，表格可以居中

## 3. 示例与效果

- 示例

        | 靠左  | 居中  |  靠右 |
        | :---- | :---: | ----: |
        | 12345 | 12345 | 12345 |

- 效果

    | 靠左  | 居中  |  靠右 |
    | :---- | :---: | ----: |
    | 12345 | 12345 | 12345 |
