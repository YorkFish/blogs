# 删除线

## 1. 用法

### <font color="Crimson">1.1</font> Markdown 向

- 类似加引号，左右各加两个"波浪号"

### <font color="DodgerBlue">1.2</font> H5 向

- 借助 *H5* 的标签 `<s>` 或 `<del>`

## 2. 说明

- “波浪键”就是 <kbd>Tab</kbd> 上面的那个按键——<kbd>`~</kbd>
- 有人也把“波浪键”称为“飘号键”
- `<s>` 标签已经过时了，但还能用

## 3. 示例与效果

- 示例

        ~~删除文本~~
        <s>s 标签过时了<\s>
        <del>del 标签也可删除<\del>

- 效果

    - ~~删除文本~~
    - <s>s 标签过时了</s>
    - <del>del 标签也可删除</del>
