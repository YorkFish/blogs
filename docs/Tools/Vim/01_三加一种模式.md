## 4. “可视” 模式

- 这个模式是基于“编辑模式”的
- 按 <kbd>Ctrl</kbd> + <kbd>v</kbd> 进入 **“可视-块”**模式——“横向选择”
- 按 <kbd>v</kbd> 进入 **“可视”** 模式——“纵向选择”
