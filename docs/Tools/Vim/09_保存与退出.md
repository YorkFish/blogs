# 保存与退出

## 1. 保存

| 命令 | 释义 |
| :---: | :--- |
| **:w** | （小写）保存 |
| **:w new_file_name** | 1. *new_file_name* 是虚指，实际用时由具体文件名代替 <br>2. 此命令意为“另存为指定文件” |

## 2. 退出

| 命令 | 释义 |
| :---: | :--- |
| **:q** | （若没做更改）直接退出 |
| **:q!** | 不保存退出，也可理解为“强行退出” |
| **:x** | （小写）没更改文件时，效果同 `:q` |

## 3. 保存并退出

| 命令 | 释义 |
| :---: | :--- |
| **:wq** | 保存并退出 |
| **:wq!** | 强行保存并退出（“文件所有者”或 “*root* 用户”专属） |
| **:x** | （小写）更改文件时，效果同 `:wq` |
| **ZZ** | （大写）同 `:wq` |
