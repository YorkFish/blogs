# kbd & to do list

## 1. kbd

### 1.1 用法

- 它是 *H5* 标签，和别的标签使用方法一致

### 1.2 说明

- 有的平台不显示边框

### 1.3 示例与效果

- 示例

        <kbd>Ctrl</kbd> + <kbd>a</kbd>

- 效果

    <kbd>Ctrl</kbd> + <kbd>a</kbd>

## 2. to do list

### 2.1 用法

- 减号 + *1* 个空格 + 一对中括号 + *1* 个空格 + 文字
    - 中括号中需要填充内容
        - 若填 *1* 个空格：代表未完成
        - 若填 *x* （小写）：代表完成

### 2.2 说明

- `x` 表示对勾
- 有些编辑器支持鼠标点击
- 有些平台不支持这个功能

### 2.3 示例与效果

- 示例

        - [ ] 吃饭
        - [x] 学习

- 效果

    - [ ] 吃饭
    - [x] 学习
