# 4. 表格

## 1. 用法

- 标题栏 + 停靠设置 + 具体内容
- 形如

    ```
    |  |  |  |				—— 标题栏
    | :--- | :---: | ---: |	—— 停靠设置
    |  |  |  |				—— 具体内容
    ...
    ```

## 2. 说明

- 停靠效果有三种
    - `:---` —— 左对齐
    - `:---:` —— 居中
    - `---:` —— 右对齐
- `-` 至少三个
- 只用 `-` 也行，默认为左对齐
- 可以偷懒，省去左右两侧的 `|`

## 3. 示例与效果

- 示例

    ```
    | 靠左 | 居中 | 靠右 |
    | :--- | :---: | ---: |
    | 12345 | 12345 | 12345 |
    ```

- 效果

    | 靠左 | 居中 | 靠右 |
    | :--- | :---: | ---: |
    | 12345 | 12345 | 12345 |
