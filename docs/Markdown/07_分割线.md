# 分割线

## 1. 用法

### <font color="Crimson">1.1</font> Markdown 向

- *3* 个或 *3* 个以上星号

    > \***

- *3* 个或 *3* 个以上减号

    > \---

- *3* 个或 *3* 个以上下划线

    > \___

### <font color="DodgerBlue">1.2</font> H5 向

- 借助 *H5* 的标签 `<hr>`

- 直接用

        <hr>

- 设置长度

        <hr width=50%>		<!-- 按百分比 -->
        <hr width=200px>	<!-- 按像素 -->

## 2. 说明

- <font color="Crimson">1.1</font> 的方法可以插入空格，但不能插入其他元素

        *** ***
