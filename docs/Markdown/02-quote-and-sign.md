# 2. 引用与标记

## 1. 引用

- 用法：`>` + `1` 个空格 + 引用内容
- 说明
    - 引用又称“块注释”
    - 可以嵌套

### 示例与效果

#### 单行

- 示例

    `> markdown`

- 效果

    > markdown

#### 嵌套

- 示例

    ```
    > 那个春天，你站在那株粉色桃树下，粉色的花丛中是你温暖的一袭红衣。让我想起《诗经》中的
    >
    > > 逃之夭夭，灼灼其华
    >
    > 后来我才知道，那一刻，你心里想的是
    >
    > > 之子于归，宣其室家
    >
    > —— 《在唐诗里孤独漫步》
    ```

- 效果

    > 那个春天，你站在那株粉色桃树下，粉色的花丛中是你温暖的一袭红衣。让我想起《诗经》中的
    >
    > > 逃之夭夭，灼灼其华
    >
    > 后来我才知道，那一刻，你心里想的是
    >
    > > 之子于归，宣其室家
    >
    > —— 《在唐诗里孤独漫步》

## 2. 标记

- 用法：用两个反引号包框住
- 说明：<kbd>\`\~</kbd> 就是 <kbd>Tab</kbd> 上面的那个“波浪键”

### 示例与效果

- 示例
    - \`标记\`
- 效果
    - `标记`
