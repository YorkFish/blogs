# 18. LaTeX

## 1. 用法

- `$$ + 编辑的内容 + $$`

## 2. 说明

### 几个名词解释

| 名称 | 释义 |
| :---: | :--- |
| TeX | 学术排版 |
| LaTe | 相当于 TeX 的简化版本；对公式编辑精细至像素级别 |
| MathJax | 相当于 LaTeX 的简化版本，用于显示数学表达式式 |
| Markdown | 不支持 TaTeX 所有的语法，对数学公式的支持比较好 |

### 注意事项

- `LaTex` 区分大小写

## 3. 示例与效果

- 示例

    ```
    $$
    \pi(x) = \int_0^x \frac{dt}{lnt} + C
    $$
    ```

- 效果

    $$
    \pi(x) = \int_0^x \frac{dt}{lnt} + C
    $$
