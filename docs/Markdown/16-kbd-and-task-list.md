# 16. 按键与任务列表

## 1. 按键

- 用法：借助 `HTML` 的标签 `<kbd>`
- 说明：有的平台不显示边框

### 示例与效果

- 示例

    `<kbd>Ctrl</kbd> + <kbd>a</kbd>`

- 效果

    <kbd>Ctrl</kbd> + <kbd>a</kbd>

## 2. 任务列表

### 示例与效果

- 示例

    ```
    - [ ] 党参
    - [x] 枸杞
    ```

- 效果

    - [ ] 党参
    - [x] 枸杞

### 说明

- `x` 表示打勾
- 有些编辑器支持鼠标点击
