# 7. 分割线

## 1. Markdown 方式

- 用法：至少三个星号
- 说明
    - 独占一行
    - 星号间可以有空格，但不推荐
    - `*` 可以用 `-` 或者 `_` 代替

### 示例与效果

- 示例  

    `***`

- 效果  

    ***

## 2. HTML 方式

- 用法：借助 `HTML` 的标签 `<hr>`
- 说明：`<hr>` 的 `width` 参数有两种方式设置长度

### 示例与效果

- 示例

    ```
    <hr>
    <hr width=50%>    <!--按百分比-->
    <hr width=200px>  <!--按像素-->
    ```

- 效果

    <hr>
    <hr width=50%>
    <hr width=200px>
