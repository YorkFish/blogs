# 9. 保存与退出

## 1. 保存

| 命令 | 释义 |
| :---: | :--- |
| `:w` | 保存 |
| `:w!` | 强行保存 |
| `:w {filename}` | 另存为指定文件 |
| `:w {path}/{filename}` | 另存为指定目录的指定文件 |

!!! tip "局部内容另存为"
    1. 进入“可视-块”模式
    2. 选择内容
    3. 输入 `:w filename` 进行局部另存为

## 2. 退出

| 命令 | 释义 |
| :---: | :--- |
| `:q` | 直接退出 |
| `:q!` | 强行不保存退出 |
| `:x` | 没更改文件时，效果同 `:q` |

## 3. 保存并退出

| 命令 | 释义 |
| :---: | :--- |
| `:wq` | 保存并退出 |
| `:wq!` | 强行保存并退出（需要权限） |
| `:x` | 更改文件时，效果同 `:wq` |
| `ZZ` | 同 `:wq` |
