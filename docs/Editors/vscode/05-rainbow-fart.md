    - 使用先进技术

## 1. 安装

1. 点击侧边栏的 `Extensions` (<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>x</kbd>)
2. 搜索 `Rainbow Fart`
3. 点击 `Install`

## 2. 使用

1. <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>p</kbd>
2. `Enable Rainbow Fart`
3. 弹出窗口，选择“允许访问”
4. 按右下角提示，点击 `Open`，浏览器会打开 `http://127.0.0.1:7777/`
5. 点击“需要授权”下方的 `Button`，听到“当”即可
