# 8. 后悔药与特殊字符

## 1. 后悔药

| 键位 | 释义 |
| :---: | :--- |
| **u** | 撤销 |
| `<C-r>` | 恢复撤销 |

## 2. 特殊字符

!!! note ""
    在使用搜索命令时，若遇到下面 `10` 个特殊字符，需要转义

| 序号 | 特殊字符 | 用法 |
| :---: | :---: | :---: |
| 01 | `.` | `\.` |
| 02 | `*` | `\*` |
| 03 | `[` | `\[` |
| 04 | `]` | `\]` |
| 05 | `^` | `\^` |
| 06 | `%` | `\%` |
| 07 | `/` | `\/` |
| 08 | `?` | `\?` |
| 09 | `~` | `\~` |
| 10 | `$` | `\$` |
