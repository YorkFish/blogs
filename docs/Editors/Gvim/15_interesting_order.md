# 15. 有意思的命令

## 1. 进入编辑模式

1. <kbd>Esc</kbd>
2. <kbd>Ctrl</kbd> + <kbd>[</kbd>
3. 自定义

    ```
    inoremap jk <Esc>
    ```

## 2. 显示光标处字符的编码

`ga`

## 3. 重选上回的选择

`gv`

## 4. 大小写转换

- 转为大写
    - 选中后按 `gU`
    - `gU` + 定位，如 `gUe`
- 转为小写
    - 选中后按 `gu`
    - `gu` + 定位，如 `gue`

- 可以结合其他命令，如
    - `gu5e`
    - `gu0`
    - `gugg`

- 翻转大小写：选中后按 `g~`

## 5. 自增与自减

- 自增：`<C-a>`
- 自减：`<C-x>`

## 6. 查看剪贴板内容

`:reg`

## 7. 复制

- `copy` 或 `t`
- `:[range]co[py] {address}`
- `:[range]t {address}`

## 8. 自动缩进

- 选中后按 `=`
