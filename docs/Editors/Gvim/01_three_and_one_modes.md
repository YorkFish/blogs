## 4. 可视模式

- 这个模式是基于“编辑模式”的
- <kbd>v</kbd> : 类似选中
- <kbd>Ctrl</kbd> + <kbd>v</kbd> : 列选择
- <kbd>Shift</kbd> + <kbd>v</kbd> : 行选择
