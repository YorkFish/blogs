# 10. 多行注释

## 方法一

| 命令 | 释义 |
| :--- | :--- |
| `:n1,n2s/^/#/g` | 多行行首加 `#`（行首用 `^`，`0` 达不到效果） |
| `:n1,n2s/^#//g` | 多行行首去 `#` |
| `:n1,n2s/^/\/\//g` | 多行行首加 `//` |

!!! note
    - `n1`, `n2` 是虚指，实际用时由具体命令或数字代替
    - `:n1,n2s/^/\/\//g` 可以偷懒成 `:n1,n2s/^/\/\/`

## 方法二

- 借助“可视-块”模式
    1. 按组合键 <kbd>ctrl</kbd> + <kbd>v</kbd>
    2. 使用 <kbd>j</kbd>、<kbd>k</kbd> 等选中需要注释的区域
    3. 按 <kbd>I</kbd>
    4. 键入 `\\` 或其他注释符
    5. 按 <kbd>Esc</kbd> (Over)
