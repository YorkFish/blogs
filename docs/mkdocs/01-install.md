## 2. 安装

- 安装命令：`pip install mkdocs`
    - 若已经装过，则不会重复安装，命令行会输出安装信息
- 查看版本命令：`mkdocs --version`

## 3. 注意事项

- 安装 `mkdocs` 前需要先安装 `Python`
- 若电脑中同时安装了 `Python2`, `Python3`，需要使用 `pip3 install mkdocs`
- `Unix/Linux` 系统，可能需要使用 `sudo pip3 install mkdocs`
- 若某些命令无效，可以试试
    - 重新设置环境变量
    - 使用 `python -m` + 相关命令
