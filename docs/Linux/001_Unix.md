# Unix

## 1. 发展历史简介

- **1965** 年
    - *MIT*，通用电气 (*GE*)，*AT&T* 的“贝尔实验室”联合开发
    - 项目名称：*Multics*
    - 目标：开发一种交互式的，具有多道程序处理能力的分时操作系统
    - 后来：贝尔实验室宣布退出

- **1969** 年
    - 美国贝尔实验室的<u>肯·汤普森</u>在 *DEC·PDP-7* 机器上开发出了 *UNIX* 系统

- **1971** 年
    - 肯·汤普森的同事<u>丹尼斯·里奇</u>发明了 *C* 语言

- **1973** 年
    - *UNIX* 系统的绝大部分源代码用 *C* 语言重写，提高了 *UNIX* 系统的可移植性
