# *RPM* 包的安装、升级与卸载

## 1. 包全名与包名

- 包全名
    - 若操作的包是没有安装的软件包时，使用包全名
    - 要注意路径
- 包名
    - 操作已经安装的软件包时，使用包名
    - 是搜索 `/var/lib/rpm/` 中的数

## 2. 安装

- `york$ rpm -ivh 包全名`

    | 选项 | 释义 |
    | :---: | :--- |
    | `-i` | 安装，*i for install* |
    | `-v` | 显示详细信息，*v for verbose* |
    | `-h` | 显示进度，*h for hash* |
    | `--nodeps` | 不检测依赖性 |

## 3. *RPM* 包升级

- `york$ rpm -Uvh 包全名 升级`

    | 选项 | 释义 |
    | :---: | :--- |
    | `-U` | 升级，*u for upgrade* |

## 4. 卸载

- `york$ rpm -e 包含`

    | 选项 | 释义 |
    | :---: | :--- |
    | `-e` | 卸载 |
    | `--nodeps` | 不检查依赖性 |
