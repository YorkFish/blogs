# *gzip*

## 1. 命令简介

- <u>压缩解压命令</u>
- 命令名称：<font color=#0099ff size=5>G</font>NU <font color=#0099ff size=5>zip</font>
- 命令所在路径：`/bin/gzip`
- 执行权限：所有用户
- 功能描述：***压缩文件***
- 压缩后文件的格式：`.gz`

## 2. 命令语法

- `york$ gzip [文件]`

## 3. 举例

- 压缩当前目录下的所有文件
    - `york$ gzip *`

### 补充

- 解压 `.rar` 需要在 *Linux* 中装 *rar for linux*
- `.zip` 在 *Windows* 与 *Linux* 中都可以直接解压
- 命令 `rm -rf *` 可以删除当前目录下所有文件，但不包括压缩包
