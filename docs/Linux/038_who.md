# *who*

## 1. 命令简介

- <u>用户管理命令</u>
- 命令所在路径：`/usr/bin/who`
- 执行权限：所有用户
- 功能描述：***查看登录用户信息***

## 2. 命令语法

- `york$ who`

## 3. 举例

    york$ who
    root	tty1	2020-01-30	11:00
    root	pts/0	2020-01-31	17:30	192.168.198.1
    ym		pts/1	2020-01-31	18:00	192.168.198.1

- 解释

    | root | tty1 | 2020-01-31 17:30 | 192.168.198.1 |
    | :---: | :---: | :---: | :---: |
    | 登录用户名 | 登录终端{本地} | 登录时间 | 登录终端地址 |

    | ym | pts/1 | 2020-01-31 18:00 | 192.168.198.1 |
    | :---: | :---: | :---: | :---: |
    | 登录用户名 | 登录终端（远程） | 登录时间 | 登录终端地址 |

- 补充
    
    - 退出：`york$ exit`
