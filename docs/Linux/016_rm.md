# *rm*

## 1. 命令简介

- <u>目标处理命令</u>
- 命令英文原意：<font color=#0099ff size=5>r</font>e<font color=#0099ff size=5>m</font>ove
- 命令所在路径：`/bin/rm`
- 执行权限：所有用户
- 功能描述：***删除文件***

## 2. 命令语法

- `rm -rf [文件或目录]`
    - `-r`：删除目录
    - `-f`：强制执行

## 3. 举例

1. 删除文件 `/tmp/yum.log`
    - `york$ rm /tmp/yum.log`
2. 强制删除目录 `/tmp/Japan/longze`
    - `york$ rm -rf /tmp/Japan/longze`
