# *rmdir*

## 1. 命令简介

- <u>目标处理命令</u>
- 命令英文原意：<font color=#0099ff size=5>r</font>e<font color=#0099ff size=5>m</font>ove empty <font color=#0099ff size=5>dir</font>ectory
- 命令所在路径：`/bin/rmdir`
- 执行权限：所有用户
- 功能描述：**删除空目录**

## 2. 命令语法

- `york$ rmdir [目录名]`

## 3. 举例

- 删除目录 *boduo*
    - `york$ rmdir /tmp/Japan/boduo`
