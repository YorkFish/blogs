# 软件包管理简介

## 1. 软件包分类

- 源码包
    - 脚本安装包
- 二进制包
    - 也称“*RPM* 包”或“系统默认包”

## 2. 源码包

### 2.1 源码包的优点

- 开源，若能力足够，可以自行修改源代码
- 可以自由选择所需的功能
- 软件是编译安装，所以更加适合自己的系统，更加稳定、效率更高
- 卸载方便

### 2.2 源码包的缺点

- 安装过程步骤较多
- 安装较大的软件集合时，容易出现拼写错误
- 编译时间较长，比二进制安装时间长
- 编译安装过程中一旦报错，新手很难解决

## 3. *RPM* 包

### 3.1 二进制包的优点

- 包管理系统简单，只通过几个命令就可以实现包的安装、升级、查询和卸载
- 安装速度比源码包安装快得多

### 3.2 二进制包缺点

- 经过编译，不再可以看到源代码
- 功能选择不如源码包灵活
- 依赖性
