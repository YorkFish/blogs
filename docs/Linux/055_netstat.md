# *netstat*

## 1. 命令简介

- <u>网络命令</u>
- 命令所在路径：`/bin/netstat`
- 执行权限：所有用户
- 功能描述：***显示网络相关信息，查询网络状态***

## 2. 命令语法

- `york$ netstat [选项]`

    | 选项 | 释义 |
    | :---: | :--- |
    | `-t` | *TCP* 协议 |
    | `-u` | *UDP* 协议 |
    | `-l` | 监听 |
    | `-r` | 路由 |
    | `-n` | 显示 *IP* 地址和端口号 |

## 3. 举例

1. 查看本机监听的端口
    - `york$ netstat -tlun`
2. 查看本机所有的网络连接
    - `york$ netstat -an`
3. 查看本机路由表
    - `york$ netstat -rn`
