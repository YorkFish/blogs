# *gunzip*

## 1. 命令简介

- <u>压缩解压命令</u>
- 命令名称：<font color=#0099ff size=5>G</font>NU <font color=#0099ff size=5>unzip</font>
- 命令所在路径：`/bin/gunzip`
- 执行权限：所有用户
- 功能描述：***解压缩*** `.gz` ***的压缩文件***

## 2. 命令语法

1. `york$ gunzip [压缩文件]`
2. `york$ gzip -d [压缩文件]`

## 3. 举例

- `york$ gunzip boduo.gz`

### *gzip* 补充

- 只能压缩文件，不能压缩目录
- 压缩完不保留原文件
