# *wall*

## 1. 命令简介

- <u>网络命令</u>
- 命令名称：<font color=#0099ff size=5>w</font>rite <font color=#0099ff size=5>all</font>
- 命令所在路径：`/usr/bin/wall`
- 执行权限：所有用户
- 功能描述：***发广播信息***

## 2. 命令语法

- `york$ wall [message]`

## 3. 举例

- 发一条内容为 *SC is an honest man!* 的广播
    - `york$ wall SC is an honest man!`
