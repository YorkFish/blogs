# *locate*

## 1. 命令简介

- <u>文件搜索命令</u>
- 命令所在路径：`/bin/locate`
- 执行权限：所有用户
- 功能描述：***在文件资料库中查找文件***

## 2. 命令语法

- `york$ locate 文件名`

## 3. 举例

- `york$ locate inittab`

## 4. 补充

- *locate* 搜索的是资料库中的信息，哪些信息并不是实时更新的
- 更新信息的命令：`updatedb`
- `/tmp` 下是临时文件，不在文件资源搜索范围之中
- 若不想区分大小写，可以使用参数 `-i`，如
    - `york$ locate -i test.py`
