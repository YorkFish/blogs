# *ping*

## 1. 命令简介

- <u>网络命令</u>
- 命令所在路径：`/bin/ping`
- 执行权限：所有用户
- 功能描述：***测试网络连通性***
- 压缩后文件格式：`.tar.gz`

## 2. 命令语法

- `ping 选项 IP地址`
    - `-c`：指定发送次数

### 补充

- 按 <kbd>Ctrl</kbd> + <kbd>c</kbd> 退出
- 主要看 packet loss弄包率

## 3. 举例

1. *ping* 一下 `192.168.1.156`
    - `york$ ping 192.168.1.156`
2. *ping* 三次 `192.168.1.156`
    - `york$ ping -c 3 192.168.1.156`
