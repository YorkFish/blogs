# 重启命令

## 1. 命令

### 1.1 回顾

- `york$ shutdown -r`

### 1.2 新命令

1. `york$ reboot`
2. `york$ init6`

## 2. 系统运行级别

| 级别（编号） | 释义 |
| :---: | :--- |
| **0** | 关机 |
| **1** | 单用户 |
| **2** | 不完全多用户，不含 *NFS* 服务 |
| **3** | 完全多用户 |
| **4** | 未分配 |
| **5** | 图形界面(*X11*) （一般大写的 `X` 代表图形界面） |
| **6** | 重启 |

### 补充

1. 修改系统默认运行级别

        york$ cat /etc/init
        # ...
        # 前面都是注释
        # ...
        id:3:initdefault:

2. 查询当前运行级别
   
        york$ runlevel
        N 3  # 前面的 N(N for Null) 表示之前的级别，后面的 3 表示当前的级别
