# 9. 将远程仓库克隆到本地

## 1. 登入 GitHub 账号

1. 在 GitHub 上选一个自己的仓库
2. 点击 <kbd>Clone or download</kbd>
3. 复制自己的地址

    ![](./imgs/09-01_copy_address.png)

## 2. 在本地打开 Git Bash

1. 选一个路径
2. 键入 `git clone <address>`

    ```bash
    York@DESKTOP MINGW64 /d/git
    $ git clone git@github.com:YorkFish/interesting.git
    ```
