# 查看历史纪录

## 1. 点击右侧的 *History*

- 每个文件都有自己的 *History*

    ![](./imgs/04-01_history_button.png)

## 2. 页面跳转

![](./imgs/04-02_history_detail.png)

## 3. 点击相应的 *commit* 标题

- 这是 *commit* 时写的“标记”
- 点击相应的 *commit* 可进入相应版本查看相应的历史
- 如，我选择了 `I am adding heap.`

    ![](./imgs/04-03_select_history.png)

- 点击后出现如下页面

    ![](./imgs/04-04_shows_the_changed.png)

- 箭头处说明第 *4* 行是该次 *commit* 时加入的
