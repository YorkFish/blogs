#  创建分支与切换分支

## 1. 创建分支

### 1.1 创建

1. 点击左上方的 `Branch: master`
2. 在输入框中填入分支名，如 *data_structure*
3. 点击下方的 `Create branch`

    ![](./imgs/05-01_create_branch.png)

- 此时，已经切到了刚刚建立的分支上

    ![](./imgs/05-02_select_branch.png)

## 2. 切换分支

### 2.1 左上角找 *Branch*

1. 点击 `Branch`
2. 选择 `Branches`
3. 选择想要切换的分支即可

- 下方两图表示：从 `data_structure` 切到 `master` 的分支
